Lists a users payment methods.  
  
*method:* `GET`  
*path:* `/v1/payment-methods`  
  
Parameters  
-----------  
  
**Body**  
  
`none`  
  
**Query**  
  
`none`  
  
Specifications  
--------------  
  
- should require authorization  
- should return a user's payment methods  
- should NOT return other users' payment methods
